## ✨ Overview

TweakyLITE is a simple Windows Utility designed to Windows tweaks, managing system services, controlling processes, and optimizing workflows

Built using **PowerShell**

---

## 🔥 Features

| - | Feature | Description |
|------|--------|-------------|
| 🖥 | **System Informations** | Monitoring System Info. |
| 🌐 | **Network Tools** | Optimizing and Enhancing Network Quality. |
| ⚙️ | **Performance Tweaks** | Optimizing some usages for perfomance quality. |
| 🧹 | **Clean Up** | Manage Storage and Bloatware. |
| 🛠️ | **Game Tweaks** | Optimizing perfomance for better gaming. |

---

### ⚠️ Disclaimer

TweakyLITE modifies Windows system settings.

Use at your own risk.
It is recommended to create a restore point before applying tweaks.
